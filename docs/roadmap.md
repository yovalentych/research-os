# MVP Roadmap (2–4 тижні)

## Тиждень 1 — База
- Створення структури проекту
- MongoDB Atlas + базові моделі
- Auth (email/password) + ролі
- Перші CRUD для Projects, Experiments, Files

## Тиждень 2 — MVP модулі
- Dashboard + Experiments + Candidates + Vault + Papers
- Канбан для кандидатів
- Vault з тегами і прив'язками
- Audit log + версії полів

## Тиждень 3 — Розширення
- Notes + Sources (публікації)
- Inventory + Finance (мінімально)
- Протоколи + шаблони

## Тиждень 4 — Dissertation flow
- Блоки дисертації (Methods/Results/Intro)
- Автозбірка контенту з експериментів/фігур
- Стабілізація і підготовка до деплою
